# How to Install Ubuntu 20.04 Alongside with Windows 10 in Dual Boot

- Follow the installation process from this [post](https://www.itzgeek.com/post/how-to-install-ubuntu-20-04-alongside-with-windows-10-in-dual-boot/) but please read the notes below before begin the installation process.
- It is recommended to allocate around 50GB to 100GB (1GB = 1024MB) disk space for Ubuntu.
- It is recommended to do automatic partitioning instead of manual partitioning when partitioning the disk at ubuntu installation process.
- If you're still in doubt to do it by yourself (especially in the Disk Partitioning step), ask a friend who is experienced in this kind of process to assist you.
